# ADR-0002: Güncelleme için electron-updater

## Context
Otomatik güncelleme ihtiyacı ve GitHub Releases ile dağıtım.

## Decision
`electron-updater` kullanımı.

## Consequences
+ Kolay entegre, GitHub ile uyumlu
- Ağ kısıtlarında başarısızlık olasılığı (runbook mevcut)

## Alternatives
- Manuel indirme/kurulum akışı
